# Assumptions & Next Steps

## Key Assumptions

| # | Assumption |
| --- | --- |
| 1 | Roster and timesheet Excel exports will maintain their current column structure from Employment Hero / KeyPay |
| 2 | Payslip PDF format is standard KeyPay output |
| 3 | Revenue data will be entered manually per location per period (no POS/accounting integration required) |
| 4 | All locations operate under the single Master Dry Cleaners entity |
| 5 | Fortnightly pay cycle aligns with roster export periods |
| 6 | The "Cost" column in roster exports is pre-calculated by Employment Hero using correct award rates |

---

## To Get Started

To proceed, we need confirmation on:

1. **Approval** of scope and budget
2. **Ongoing access** to roster, timesheet, and payslip exports for testing and refinement
3. **Revenue data** — preferred format for entering weekly/fortnightly revenue per location
4. **Wage % targets** — desired target percentage per location (can be configured later)
